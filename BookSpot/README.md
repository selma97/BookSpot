# Book Shopping Cart MVC

Work in progress ASP.NET Core MVC project for an online bookstore application.

## Description
This project is a basic e-commerce web application focused on implementing shopping cart functionality. It is being developed as part of learning and practicing ASP.NET Core MVC, Entity Framework, and database integration.

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Identity Core (Authentication)
- Bootstrap 5

## Features Implemented
- Display of books/products
- Basic application structure (MVC pattern)
- Add to cart functionality
- View cart (shopping cart overview)

## Current Status
This project is currently a work in progress. The main focus so far has been on implementing core e-commerce features, especially shopping cart logic.

## Planned Features
- Checkout functionality
- Order processing
- User authentication and authorization
- Admin panel for managing books and orders
- UI/UX improvements

## How to Run
1. Clone the repository
2. Configure the connection string in `appsettings.json`
3. Apply migrations / set up the database
4. Run the application

## Note
This is a work in progress project created for learning purposes and to improve skills in ASP.NET Core MVC development.
